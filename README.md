# Client Website Starter

A starter Astro template for building client websites quickly using a reusable system

This starter template shows how to build client websites in minutes using one project and a simple configuration file.
It’s perfect for freelancers or developers who want to create multiple client websites without rebuilding the same layout repeatedly.

git clone https://github.com/yourusername/client-website-starter.git

cd client-website-starter

pnpm install

pnpm dev
